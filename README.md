# Fatigue Detection System 😴🚗

## 📌 Overview
This project detects drowsiness using real-time webcam input.  
It uses MediaPipe Face Mesh to track eye landmarks and calculate eye openness.  
If the eyes remain closed for a certain duration, an alert sound is triggered.

---

## 🚀 Features
- Real-time face and eye detection
- Eye openness calculation
- Drowsiness alert system 🔊
- Live graph visualization 📊
- Data logging to CSV file

---

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy
- Pandas
- Matplotlib
- Pygame

---
